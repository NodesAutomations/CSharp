### Overview
- It's stripped down version of AUTOCAD without any UI, this allow use to use autocad using command line, autocad will just load in seconds.
- But it has almost all functionality that we need for most of regular use
- It's very much suited for any batch operation
- Path : `C:\Program Files\Autodesk\AutoCAD 2015\accoreconsole.exe`
