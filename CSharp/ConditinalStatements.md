### IF Else
 ```csharp
      //IF
          int a = 10;
          int b = 20;
          if (a > b)
          {
              Console.WriteLine("A is bigger");
          }
          else
          {
              Console.WriteLine("B is bigger");
          }
```

### Switch

```csharp
int n = 5;
switch (n % 2)
{
    case 0:
        System.Console.WriteLine("Even");
        break;
    default:
        System.Console.WriteLine("Odd");
        break;
}
```
